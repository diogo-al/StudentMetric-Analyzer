# StudentMetric-Analyzer (DAAMTReader)

A Java-based data processing engine designed to parse, aggregate, and analyze student records from disparate data sources.

## 🚀 Key Features
* **Multi-Source Data Parsing:** Reads and synchronizes information from separate files (student IDs/names and test scores).
* **Automated Aggregation:** Matches records based on unique identifiers to provide a consolidated view of student performance.
* **Real-time Analytics:** Calculates averages and formats data directly in the console for immediate reporting.
* **Error Handling:** Robust logic to handle data mismatches or missing entries during the parsing process.

## 🛠️ Technologies
* **Language:** Java
* **Concepts:** File I/O, Data structures, String manipulation, and Algorithm optimization.

## 💡 Practical Application
While this project started as a tool to manage academic records during my Computer Engineering degree at ISCTE, it demonstrates core principles used in **Security Log Analysis**: extracting meaningful insights from raw, unstructured data through automation.
